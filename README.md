# Weather-Prediction-System
空氣質量監測系統

# 使用技術：
  Python、Django、pandas、numpy、LSTM
  
# 簡介：
  可从pm25.csv中讀取數據訓練並做預測
# 未來改進 :
  可改成使用預訓練的模型做訓練取代每次點擊都要訓練
